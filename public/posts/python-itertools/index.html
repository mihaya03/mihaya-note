Python itertools

chains

  • 複数のイテラブルを1つのイテラブルに連結する
    • 全要素を一度に保持する必要がないためメモリ効率が向上する
from itertools import chain

iterable1 = [1, 2, 3]
iterable2 = ['a', 'b', 'c']
iterable3 = (4, 5)
combined = chain(iterable1, iterable2, iterable3)

for item in combined:
    print(item)
# 1 2 3 a b c 4 5
